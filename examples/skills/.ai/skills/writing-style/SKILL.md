## Writing Style Guidelines

- **Be concise** — prefer short sentences; cut any word that does not add meaning
- **Use active voice** — "the function returns X" rather than "X is returned by the function"
- **Favour plain English** — avoid jargon unless the audience expects it
- **One idea per paragraph** — start a new paragraph when the topic shifts
- **Use numbered lists for steps** and **bullet lists for non-ordered items**
- **Spell out acronyms on first use** followed by the abbreviation in parentheses
